9 Summary and Conclusions

 

This report described the features required of an ARIES style log manager in an RDBMS environment. It then went on to describe the interface and implementation of various parts of the log manager and the simplifying assumptions made in the beginning and along the way. After outlining the test procedure, the report concluded with future work that would enhance the log manager and suggested a few enhancements to Minirel itself.
