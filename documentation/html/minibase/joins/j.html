Join Algorithms

Several join methods have been implemented, although only two have been integrated into Minibase. Each join method is implemented as an iterator.

Boldface indicates the code is integrated into Minibase:

  • Simple Nested Loops
  • Block Nested Loops
  • Index Nested Loops
  • Sort Merge
  • Simple Hash
  • Grace Hash
  • Hybrid Hash

The interface for the join algorithms is available here.
