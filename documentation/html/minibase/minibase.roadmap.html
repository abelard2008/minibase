Roadmap of the Minibase Distribution

  • COPYRIGHT - Copyright Notice and User Agreement.
  • INSTALL - Installation instructions.
  • CONFIG - Configuration file (needs to be set up before build).
  • Makefile - The top level makefile.

  • documentation/ - The minibase documentation.
  • libs/ - Minibase libraries.
  • include/ - Minibase include files.
  • bin/ - Minibase binaries.

  • makefiles/ - Common Makefiles.
  • library/ - Source code for Minibase Library.
  • programs/ - Source code for Minibase "main" and tools.
  • packages/ - Tk/Tcl libraries.
